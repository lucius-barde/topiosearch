# topiosearch

TopioSearch - Search tool for Swiss French words - Outil de recherche de mots suisse romands

Install procedure:
1. Install node and npm (for example, via nvm)
2. `npm install -g @vue/cli`
3. `npm install`
4. `npm run build`
5. Execute the app in the /dist folder.
